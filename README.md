SCHOOL MANAGEMENT SYSTEM
 

A Simple school management system built using python and oops (object oriented programming) , used JSON to store the data
 

## 📌 Features

### Student Management
- Register new students
- Prevent duplicate roll numbers
- Validate email addresses
- Add grades for different subjects
- View student details and academic performance
- Calculate average marks automatically

### Teacher Management
- Register new teachers
- Prevent duplicate employee IDs
- Validate email addresses
- Store subject information
- View teacher details
